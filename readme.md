# MarkDownEditor

## これは何？
- ブラウザ上で動作するマークダウンエディタ。「boostnote」というデスクトップアプリや「codesandbox」を模倣した。
- 趣味と勉強用で作った

## 使用したもの
- html
- css
- JavaScript
- PHP

### フレームワーク
- Vue.js
- Laravel
